# dy_auto_comment
个人用dy自动点赞评论工具

**使用前请确保你下载/更新了Google Chrome浏览器:)**
## 2026/02/28
## 版本2.2.0
1.新增逻辑:如果抖音刷视频界面出现了验证码，会自动暂停程序

2.修复了偶现的豆包回复过长导致被误判为出现验证码(这会导致程序自动暂停)的bug

3.修复了偶现的评论区未被正确打开导致本次评论失败的bug

4.修复了打开抖音官网时不会自动跳转到’推荐’界面的bug(此bug最近才出现，不清楚是什么原因)

## 版本1.0.0
最开始的版本，版本功能如下:

每次将随机观看视频/直播[10, 30]秒
