# dy_auto_comment
个人用dy自动点赞评论工具

**使用前请确保你下载/更新了Google Chrome浏览器:)**
## 2025/12/27
## 版本1.2.0
1.新增逻辑:如果检测到验证码出现，会自动暂停程序

2.修复了启用’评论失败处理’功能后，该功能仍执行异常的bug

## 2025/12/16
## 版本1.1.0
1.新增了一些可修改的配置，以优化自动点赞或者评论视频/直播的逻辑，你可以在_internal/config.json中修改这些配置，相关配置说明请参考’config.json文件说明.docx’这个文档

2.新增了键盘事件监听，相关说明请参考’键盘事件监听.docx’这个文档

## 2025/12/15
## 版本1.0.0
最开始的版本，版本功能如下:

每次将随机观看视频/直播[10, 30]秒

在随机观看[20, 30]个视频/直播后会进行点赞和评论(只会对点赞数>=1w的视频进行点赞和评论)
